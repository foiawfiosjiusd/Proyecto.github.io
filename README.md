<section>
    <h2> Descripción del proyecto</h2>
    <p>
        Esta aplicación web permite a estudiantes y aspirantes consultar información relevante
        sobre una universidad, incluyendo la oferta académica, costos de titulación y modalidades disponibles.
        Está diseñada para mejorar la accesibilidad y comprensión de la información institucional.
    </p>
</section>

<section>
    <h2> Objetivo</h2>
    <p>
        Facilitar el acceso a información académica mediante una plataforma clara, organizada e intuitiva,
        apoyando la toma de decisiones de los estudiantes.
    </p>
</section>

<section>
    <h2> Funcionalidades</h2>
    <ul>
        <li>Visualización de licenciaturas disponibles</li>
        <li>Consulta de costos de titulación</li>
        <li>Información sobre modalidades de titulación</li>
        <li>Sección de preguntas frecuentes (FAQ)</li>
        <li>Integración con Google Maps</li>
        <li>Diseño responsive</li>
    </ul>
</section>

<section>
    <h2> Enfoque técnico</h2>
    <ul>
        <li>Desarrollo frontend estructurado</li>
        <li>Diseño responsive con CSS</li>
        <li>Integración de APIs externas (Google Maps)</li>
        <li>Buenas prácticas de UX/UI</li>
    </ul>
</section>
